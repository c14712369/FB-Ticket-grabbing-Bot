# 排球輕鬆玩-自動留言機器人

這是一個專為 Facebook 社團與相簿搶留言設計的 Tampermonkey 腳本，特別優化了排球活動的搶票/搶留言流程。

## 🌟 功能特點
- **排球輕鬆玩專屬優化**：自動計算並預設下週一日期，一鍵設定關鍵字。
- **OCR 影像文字辨識**：全自動掃描相簿封面，尋找特定日期與時段並自動點擊進入。
- **持久化設定**：自動儲存地點濾鏡、留言內容、啟動時間，重開瀏覽器不遺失。
- **高精準計時器**：動態追蹤系統時間，解決瀏覽器節流導致的秒數卡死。
- **自動監控跳轉**：在相簿列表頁面設定時間，準時自動刷入目標相簿。
- **快速留言**：進入相片大圖模式後，自動以毫秒級速度填寫留言並按 Enter 送出。

## ⚙️ 安裝方法
1. 安裝瀏覽器擴充功能 [Tampermonkey (油猴)](https://www.tampermonkey.net/)。
2. 建立新腳本，將 `FB_Ticket_Bot.user.js` 的完整內容貼上並儲存。
3. **重要**：第一次執行時若跳出「跨網域要求」，請點擊「**總是允許全部**」以啟用 OCR 影像辨識 API。

## 📖 使用說明
1. **進入相簿列表頁面**：開啟 Facebook 社團或活動的「相簿」分頁（網址需含 `/media`）。
2. **設定參數**：
   - **啟動時間**：設定搶票開始的時間（如 `18:59`）。
   - **地點與留言**：填寫您的過濾關鍵字與要發出的留言內容。
3. **啟動監控**：點擊「▶ 啟動監控」。左上角將顯示倒數計時，時間到後腳本將自動搜尋相簿並執行後續動作。
