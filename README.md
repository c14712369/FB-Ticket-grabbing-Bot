# FB Ticket Grabbing Bot (V21.0)

這是一個用於 Facebook 搶票的 Tampermonkey 腳本。

## 功能特點
- **GUI 控制面板**：直觀的圖形介面，無需修改代碼即可調整參數。
- **動態參數調整**：即時修改關鍵字、地點濾鏡、留言內容與點擊座標。
- **自動監控**：自動重新整理頁面，直到發現符合關鍵字的目標。
- **快速留言**：鎖定目標後自動填寫留言並送出。
- **座標導引**：在頁面上顯示準心，輔助設定點擊位置。

## 安裝方法
1. 安裝 [Tampermonkey](https://www.tampermonkey.net/) 瀏覽器擴充功能。
2. 點擊 `FB_Ticket_Bot.user.js` 並複製內容。
3. 在 Tampermonkey 中新增腳本並貼上。

## 使用說明
1. 開啟 Facebook 目標社團或活動頁面。
2. 點擊右下角的控制面板進行設定。
3. 點擊「啟動監控」開始自動搜索，或點擊「強制搶票」手動執行。
